# Vibe Coding 周末的日程和工作台

自从开始 Vibe Code, 我的典型的周末一天日程 be like:

起床 → Vibe Coding → 跑步 → 洗澡 → Vibe Coding → 吃早餐 → Vibe Coding → 吃午饭（外卖） → Vibe Coding → 午休 → Vibe Coding → 游泳 → Vibe Coding → 吃晚饭（外卖） → Vibe Coding → 睡觉
昨天除了拿外卖就没出过酒店门，因此得去跑步和游泳来消减一天久坐对身体带来的负面影响，早上跑步 3 公里，下午游泳 40 个来回约 800 米。

![vibe-code-workstation](/flows/vibe-code-workstation.jpg)

但周末依然让我兴奋，因为这是一周内生产力最高的时候，图片是我的周末办公桌面（图片拍摄于上个月）：

1. 三台电脑一台外接显示器：一台工作电脑预备紧急会议和写材料；一台个人电脑+外接显示器作为 Vibe Coding 主力机；一台旧个人电脑装 OpenClaw（目前还只当它做玩具，后面再深入研究）。
2. Keychron K9 Pro 蓝牙键盘是我最喜欢的输入设备，不管去哪里都会带上它。
3. 绿色圆形设备是 B&O A1 蓝牙随身音箱，结合 Apple Music 提供 Vibe Coding 的“氛围音乐”。当前主要是交响乐+摇滚。本周末的“氛围音乐”是 Kate Bush + Pink Floyd + Beethoven.

经过[近 700 个 Commit](https://github.com/hutusi/amytis/commits/main/) 的打磨，第一个 Vibe Coding 作品 [Amytis](https://github.com/hutusi/amytis) 终于达到发布标准了（其实已经实际应用了），感觉从未有过最近几个月的生产力。
