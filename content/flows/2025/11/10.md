AI Coding 或者说 Code Agent 赛道真的是竞争激烈。目前看来主要有两种形态，一种是 IDE Agent，典型的代表是 Cursor, 另外还有 Windsurf, 国内的有 Trae (字节), 通义灵码（阿里），还有开源的 Cline, Roo Code, 以及 Kilo Code, 后面这三个开源的关系，分别是 Roo Code 从 Cline 库中 fork 出来，Kilo Code 又是从 Roo Code 中 fork 出来。我是看 GitLab CEO 的 linkedin 信息才了解 Kilo Code 的，发现他于今年 3月与别人一起创办了 Kilo Code, 打着的口号是 "the world's best open-source agentic development tool!"
另外一种形态是CLI形式的，如 Anthropic 的 Claude Code, OpenAI 的 CodeX, Google 的 Gemini CLI. 有意思的是第二种都是大模型公司推出的产品。
