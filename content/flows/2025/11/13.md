TIL:     
Claude 最近发布了 Agent Skills 的能力, Claude Code 也支持。
Agent Skills 里包含 instructions, scripts, resources. 感觉是扩展了 Claude 的能力，类似于 SubAgent，Skill 存在一个文件夹中，里面包含了 Skills 的描述，主要是 SKILL.md 文件，描述了主要的 instructions. 一个Skill 里包含三种内容，在被加载时可以分三层：
1. Metdata （总是被加载，描述在 SKILL.md 顶部）
2. Instructions （在被触发时加载，描述在 SKILL.md 的 Metadata 后）
3. Resources 和 Scripts （在需要时加载，存储在其他文件中）
