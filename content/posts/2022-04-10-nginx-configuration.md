Ф┴⌠Е╪─ Nginx Е╝┴Хё┘Г⌡╝Е╫∙Е▐╞Д╩╔Г°▀Е┬╟ Nginx Г └И┘█Г╫╝Ф√┤Д╩╤ `nginx.conf` О╪▄Д╦─Х┬╛Е°╗ `/etc/nginx` Х╥╞Е╬└Д╦▀Ц─┌Nginx И┘█Г╫╝Х└ Ф°╛Д╦╜ `#` Е╪─Е╓╢Д╦╨ФЁ╗И┤┼О╪▄Е┘╤Д╩√Д╦╨И┘█Г╫╝Ф▄┤Д╩╓О╪┬Nginx Ф°╞Х╞╜Д╦╨ directiveО╪┴Ц─┌

Nginx И┘█Г╫╝Ф√┤Д╩╤Д╦╩Х╕│Е┬├Ф┬░Д╦┴Е²≈(Nginx Ф°╞Х╞╜Е▐╚ context)О╪ Е┘╗Е╠─Е²≈(main context)Ц─│events Е²≈Ц─│http Е²≈Ц─┌events Е²≈ Е▓▄ http Е²≈И┐╫Х╒╚Е▄┘Е░╚Е°╗ `{}` Д╦╜О╪▄Ф╡║Ф°┴Ф╡║Е▄┘Е░╚Е°╗ `{}` Д╦╜Г └И┘█Г╫╝Ф▄┤Д╩╓Е╟╠Ф≤╞Е┘╗Е╠─Е²≈Г └Е├┘Е╝╧Ц─┌

Е┘╗Е╠─Е²≈Д╦╜Д╦╩Х╕│Ф≤╞Д╦─Д╨⌡ Nginx Ф°█Е┼║Е≥╗Г └Е┘╗Е╠─И┘█Г╫╝Ф▄┤Д╩╓О╪▄Е╕┌ log Ф≈╔Е©≈Е╜≤Ф■╬Д╫█Г╫╝Ц─│Nginx PID Ф√┤Д╩╤Х╥╞Е╬└Ц─│Nginx Х©░Х║▄ worker Х©⌡Г╗▀Ф∙╟Г╜┴Ц─┌Е╕┌Д╦┼Д╬▀Д╦╜Д╩▌И┘█Г╫╝Ф√┤Д╩╤Е╪─Е╓╢Е┬╟ `events` Е²≈Д╧▀И≈╢Г └Е├┘Е╝╧Д╦╨Е┘╗Е╠─Е²≈О╪┬Е╓╖И┐╗Е┬├Х╒╚ФЁ╗И┤┼О╪┴Ц─┌

Server Е²≈Г └Е┘╦Е·▀И┘█Г╫╝Е╕┌О╪

`listen` Ф▄┤Д╩╓Х║╗Г╓╨Ф°█Е┼║Г⌡▒Е░╛Г └IP Е▓▄Г╚╞Е▐ёО╪▄ `default_server` Х║╗Ф≤▌Х©≥Д╦╙ server Е²≈Ф≤╞И╩≤Х╝╓Г └Ф°█Е┼║Ц─┌

### location Е²≈И┘█Г╫╝
